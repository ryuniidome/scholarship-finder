# Scholarship Finder プロジェクト

## 概要
留学を希望する日本人学生が、自分に合った奨学金を診断形式で探せるWebサービス。

## ターゲットユーザー
日本人学生（学部・大学院）で海外留学を検討している人

## 主な機能
1. 診断UI：質問に答えると該当奨学金が表示される
2. 奨学金データベース：条件・金額・締切などを管理
3. 変更監視スクリプト：奨学金サイトの更新を検知して通知

## 対象奨学金カテゴリ
- 政府系（JASSO、トビタテ！留学JAPANなど）
- 民間財団（ロータリー財団、各種財団）
- 留学先国・大学が提供するもの

## 技術スタック
- フロントエンド：HTML / CSS / JavaScript
- データ：JSON（scholarships.json）
- ホスティング：Netlify（無料）
- 監視スクリプト：Python

## ファイル構成（予定）
```
scholarship-finder/
  index.html          # 診断UI
  style.css
  app.js              # 診断ロジック
  data/
    scholarships.json # 奨学金データベース
  scripts/
    monitor.py        # 変更監視スクリプト
  CLAUDE.md
```

## 診断の質問項目（案）
- 現在の学年・学位（学部 / 修士 / 博士）
- 留学先の地域・国
- 留学の目的（語学 / 研究 / 学位取得）
- 留学期間
- 専攻分野
- 日本国籍かどうか
